What's this?

Nonograph is for writing things you want to exist somewhere permanent without that somewhere being attached to you. A thought, an essay, a rant, class notes, up to you. You write it, you get a link. Share the link with whoever you'd like, or keep it to yourself.

The service and software are designed to last. Your pages are stored as plaintext markup, no proprietary formats. Being a simple, lightweight, open-source project means it should last a long time with very little funding.

Principles

  • No ads, no monetization, no investors
  • Posts cannot be edited or deleted after publishing
  • No accounts, no tracking, no analytics
  • Open to all, no political or social biases
  • Open source, self-hostable

Source code

It's entirely open source. The code is at github.com/du82/nonograph. You can read it, run it yourself, and host for others easily.

Formatting

Type / for a menu of formatting options, or use markdown directly. Add .md to any post URL to see the raw source.
